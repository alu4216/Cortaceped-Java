Proyecto cortacesped.

Codigo en Java, simulando un cortacesped. 
En este código se implementan el algoritmo de búsqueda en profundidad (DFS) y el algoritmo de búsqueda A*.
Al ejecutar el programa se puede elegir en diseñar un mapa aleatorio con diferentes obstaculos y tipos de cesped a cortar por el agente o crearlo manualmente con una serie de indicaciones.


Guía de Eclipse:

Para empezar hay que descargar e instalar eclipse:
http://www.eclipse.org/downloads/

Una vez instalado eclipse, se crea en mis documentos una carpeta, por defecto se llamara workspace.
En esta carpeta es donde meteremos nuestro programa llamado cortacésped, es decir, la carpeta del proyecto la pegamos dentro del directorio workspace.

Abrimos eclipse
Vamos a file, Import  ,general - Existing Project into workspace , le damos a conituar , seleccionamos el direcctorio en Browse  y le damos a finish. 
Una vez hecho esto, ya el proyecto esta cargado en el eclipse y listo para ser ejecutado
